R package blindrecalc for blinded sample size recalculation with internal pilot studies for a depression trial

Introduction to the package blindrecalc

  • R package for blinded sample size recalculation
  • Authors: Lukas Baumann, Maximilian Pilz and Meinhard Kieser
  • Published: March 2022
  • Features of the package:
    • Blinded sample size recalculation for Chi square, Farrington Manning, Student t test
    • Computation of the actual type I error rate for different values of the nuisance parameter
    • Calculation of the achieved power under a given set of nuisance parameters
    • Sample size distribution for different values of the nuisance parameter
